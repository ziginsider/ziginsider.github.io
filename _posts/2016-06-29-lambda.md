Вот так можно организовать обратный цикл с помощью лямбда-выражения и рекурсии:

{% highlight csharp %}
Func<int, int> fc = null;
fc = i =>
{
  Console.WriteLine(i);
  return i == 0 ? 0 : fc(--i);
};
fc(10);
{% endhighlight %}
